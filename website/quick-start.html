Quick Start

Install Synth and run the canonical governance pipeline in under five minutes.

1. Install

git clone https://github.com/synth-dev/synth-v2.git cd synth-v2 npm install

2. Run the governance pipeline

npm run govern

This builds the project, runs the full test suite, verifies replay, runs adversarial audits, and generates a proof artifact in proof/.

3. Run the operator journey

npm run test:operator-journey

This automates the full Idea → Mission → Planning → Approval → Genesis → Execution → Replay → Documentation → Done journey.

4. Try an example

cd examples/todo npm run govern

Each example is a standalone project that demonstrates a real Synth mission.

5. Read the operator guide

Continue with Getting Started to learn how to chart your first mission.
