A collection of tested prompts for operating Synth from an AI agent.

## How to use

Copy the prompt text, fill in the placeholders, and send it to your AI assistant. Each prompt includes expected behavior and safety rules.

## Contributing

New prompts should be added through an Expedition. Test the prompt with at least two different AI assistants before submitting.
