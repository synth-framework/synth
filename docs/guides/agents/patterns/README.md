# Agent Patterns

## Purpose
Reusable behavioral patterns for AI agents operating within Synth.

## Audience
AI agents.

## How to Use

Each pattern defines:
- **Name** — What the pattern is called
- **Context** — When to use it
- **Problem** — What it solves
- **Solution** — How to apply it
- **Consequences** — What happens when you use it
- **Examples** — Concrete usage
