# Operator Domain

## Purpose
Teach operators how to work with Synth using only the seven public concepts. This documentation contains practical guides, examples, and best practices for day-to-day operation.

## Audience
Operators, developers, product owners — anyone who interacts with a running Synth system.

## Public Vocabulary

All operator docs use the seven public concepts: Mission, Expedition, Evidence, Plan, Event, State, and Replay. See the [Public Vocabulary](../reference/public-vocabulary.md) for definitions.
