# Synth Example: Personal Blog

A content publishing example that demonstrates documentation generation from a Synth mission.

## Mission

Build a personal blog for publishing articles.

## Expedition

**Article Pipeline** — Implement article creation, editing, and publishing flow.

## Objectives

- Implement create draft endpoint
- Implement publish article endpoint

## Run

```bash
npm run govern
```

## Expected Results

- Replay-consistent event log.
- Proof artifact in `proof/`.
- Generated documentation in `docs-generated/`.
