# Synth Example: Polyglot Service

A polyglot example that demonstrates language-agnostic repository analysis in Synth.

## Mission

Build a service with components in multiple languages.

## Expedition

**Language Boundaries** — Define clear contracts between the Go, Python, and Node components.

## Objectives

- Define the shared service contract
- Implement language-specific adapters for the contract

## Run

```bash
npm run govern
```

## Expected Results

- Replay-consistent event log.
- Proof artifact in `proof/`.
- Generated documentation in `docs-generated/`.
