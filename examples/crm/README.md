# Synth Example: Customer CRM

A business-object example that demonstrates adapter-style integration patterns in Synth.

## Mission

Build a customer relationship management system.

## Expedition

**Contact Management** — Implement contact creation and interaction tracking.

## Objectives

- Implement add contact endpoint
- Implement log interaction endpoint

## Run

```bash
npm run govern
```

## Expected Results

- Replay-consistent event log.
- Proof artifact in `proof/`.
- Generated documentation in `docs-generated/`.
